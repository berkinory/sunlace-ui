# sunlace

A minimal, modern React component library built on Base UI and Hugeicons.

## install

```bash
npx shadcn@latest add https://sunlace.dev/r/<component>.json
```

browse components at [sunlace.dev](https://sunlace.dev)

## llm-friendly

```bash
curl https://sunlace.dev/llms.txt
```

## stack

- React 19
- Base UI
- Hugeicons
- Tailwind CSS v4
- shadcn CLI
